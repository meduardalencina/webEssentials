input radio

É uma coleção de Radio Buttons (Botões de Radio) que descrevem uma série de opções que so podem ser marcadas uma vez

o Input radio nada mais é que uma opção que pode ser marcada ou desmarcada, e possui como característica um pequeno círculo ao lado do texto.

Um radio button geralmente é utilizado em grupo (radio groups), ou seja, um conjunto de opções relacionadas, e somente um elemento do grupo pode ser selecionado por vez (escolher uma opção automaticamente desmarca as outras).
